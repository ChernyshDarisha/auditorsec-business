## **Logistics Management Checklists**

### **In-Person Workshop Setup**
**2 Weeks Before:**
- [ ] Book conference room (6-8 person capacity)
- [ ] Reserve AV equipment (projector, screen, laptop connection)
- [ ] Order catering (breakfast pastries, lunch, afternoon snacks)
- [ ] Purchase supplies (whiteboards, markers, sticky notes, flip charts)
- [ ] Confirm parking availability for attendees
- [ ] Send calendar holds to all participants

**1 Week Before:**
- [ ] Distribute prep materials package
- [ ] Confirm catering order and dietary restrictions
- [ ] Test AV equipment and room setup
- [ ] Prepare physical materials packets
- [ ] Send reminder with building access instructions
- [ ] Confirm final headcount

**Day Of:**
- [ ] Arrive 30 minutes early for setup
- [ ] Test all technology (projector, wifi, etc.)
- [ ] Arrange room layout (U-shape or collaborative)
- [ ] Set up materials stations
- [ ] Prepare welcome refreshments
- [ ] Have backup supplies available

### **Virtual Workshop Setup**
**2 Weeks Before:**
- [ ] Test video conferencing platform capacity
- [ ] Set up digital whiteboard account (Miro/Mural)
- [ ] Create breakout room structure if needed
- [ ] Design virtual workshop templates
- [ ] Send calendar invites with all connection details
- [ ] Plan engagement activities for virtual format

**1 Week Before:**
- [ ] Send tech check meeting invites (optional 15-min sessions)
- [ ] Distribute digital workshop materials
- [ ] Create shared folder access for all participants
- [ ] Test recording capabilities and permissions
- [ ] Prepare virtual icebreakers and engagement tools
- [ ] Send connection instructions and backup contact info

**Day Of:**
- [ ] Start meeting 15 minutes early for tech checks
- [ ] Have backup facilitator/tech support available
- [ ] Test all digital tools and screen sharing
- [ ] Prepare engagement polls and interactive elements
- [ ] Have phone numbers ready for connection issues
- [ ] Monitor participation and engagement levels

## **Workshop Day Execution Framework**

### **Workshop Flow Management**
**Opening (15 minutes):**
- [ ] Welcome and introductions
- [ ] Workshop objectives review
- [ ] Ground rules and decision-making process
- [ ] Technology/logistics check
- [ ] Agenda confirmation

**Each Session Block:**
- [ ] Clear start/end times communicated
- [ ] Objectives stated at beginning  
- [ ] Progress tracked against agenda
- [ ] Action items documented in real-time
- [ ] Decisions confirmed before moving forward

**Closing (15 minutes):**
- [ ] Recap key decisions and outcomes
- [ ] Confirm implementation assignments
- [ ] Schedule follow-up meetings
- [ ] Collect feedback on workshop process
- [ ] Distribute next steps communication
